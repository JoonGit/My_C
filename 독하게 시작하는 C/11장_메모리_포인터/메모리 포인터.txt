메모리 특징(구조 + 관리) ->
지정 -> 직접기정   -> 변수선언 -> 변수선언(메모리)
      -> 간접지정

메모리 지정은 자동화 되어있다
변수선언시 메모리확보 후 주소를 컴파일러가 지정하게 된다

메모리 저장 방식
Byte order 형식으로 저장한다
주소 번시장 낮은쪽이 왼쪽으로 가는것을 Litttle Endian 이라고 한다
반대 개념엔 Big Endian이 있다

ASLR(ddress Space Layout Randomization)
메모리 해킹 방지를위해
프로그램을 실행 할때마다 매번 주소영역이 변경 된다

변수 = '메모리'로 구현 &주소 -> Compile time 연산(CPU 소모를 계속하지 않는다)
