C언어의 메모리는 virtual Memory(가상메모리)이다

C언어에서 사용하는 대표 메모리 4가지
동적할당
1. stack -> 1MB, 스코프
2. Heap -> 자유영역, 32bit 싱글쓰레드 기준 약 1.7GB 정도, 읽기, 쓰기 실행 가능, 
PE image역역에
3. text(code)
4. Data - 읽기,쓰기 : 전역변수, 정적변수
	- 읽기 : 문자열 상수

메모리는 os를 통해서 제어가 된다
필요한 만큼 os에서 할당 받는다(없는게 생기는게 아니다)
할당 받은만큼 메모리를 사용후 os에 반납한다

일반 적으로 사용하는 변수는 자동변수 이기에 할당, 반납이 자동으로 이루어 진다

메모리 동적 할당 및 해제
원하는 만큼의 메모리 크기를 malloc 으로 할당 받고
사용후 free로 반환한다
malloc으로 메모리를 할당 받으면 fd 로 메모리의 경계 선이 생긴다
이 fd는 메모리의 크기가 오버가 되었는지 확인하는 용도이다
