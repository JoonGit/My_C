Unix는 C언어로 만들어 졌다고 알고 있다 자세히 알면
Unix는 1969년도에 B언어(커닝 핸리)로 초에 만들어 졌다 그러나 실패 했다
데리스 리치가 C언어(1971)를 만들고 Unix에 80%만들었다
B언어 -> C언어 가 되면서 자료형이 추가됐다는 점이 다른점이다

자료형 : 메모리에 저장된 정보를 해석, 일정길이의 정보를 해석하는 일

B언어의 매뉴얼 중 첫번째 예제가 Hello word 이다
B언어가 사라지고 C 언어가 이를 가져가게 되었다

C 언어 표준 = C99 기준으로 학습한다

main()의 시작이 프로그램의 시작이며
main()의 종료는 프래그램의 종료이다

main() -> printf()
main(caller, 호출자)에서 call하여 printf()를 호출한다
printf(피호출자, callee) 응답하게 된다
응답의 경우 printf( ) 괄호 안에 매개변수 = 신인수를 넣어 호출하게 된다
" " 괄호 안에 이게 들어가 있을 경우 문자(배)열이 된다
