배열
자료형이 인접하게 여러게 나열되어 있는 자료형이다

자료형이 [ x ] x개 만큼 나열되있는 자료형이다

- 배열의 내부
메모리 안에 자료형이 인접하게 여러게 index로 나열되어 있다

일반 변수는 식별자를 통해 메모리를 만들고 값을 넣어 주소에 저장하는데
배열의 이름은 주소이다
여러 자료형이 모여있지만 각각에 자료형 마다 주소 분리 되어 있지 않고 첫번째 자료형의 메모리 위치가 기준이 되어 index 형태로 배열의 주소들이 인접해 있다

정렬은 최대/최소값, 비교, 교환을 할수 있다는 기본능력이다

문자(배)열은
char가 여러게 묶여있는 것이가
char메모리 하나 하나에 값이 담겨 있으며 그게 모여 한 단어 처럼 보인다

문자열 -> 가변길이
배열 -> 고정길이
고정된 배열의 길이를 넘어가게 되면 overflow가 된다 
그렇기에 배열의 길이를 넉넉히 하는게 좋다
문자열 끝에 \0이 있는 이유
1. 문자열의 끝을 알려준다
2. 채워진 배열과 빈 배열의 구분하기 위해서 이다
그렇다면 \0의 자리까지 배열의 길이를 개산해야 하는가?
배열이 딱 맞다면 생기지 않는다
