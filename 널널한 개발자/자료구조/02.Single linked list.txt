Linked List = 컨테이너(자료를 담는다)
- 연결 리스트는 여러 구조체 인스턴스를 체인처럼 줄줄이 포인터로 연결한 자료구조이다
- 연겔에 사용된 포인터 숫자가 한 개 이고 자기 다음을 라리키는 것이 특징이다

연결 리스트 코딩 요령
- 구조체 배열로 테스트 한다.
- 연결 순서를 임의로 변경해본다.
- 리스트 출력은 별도 함수로 분리한다.
- 디버거로 노드 하나씩 따라가며 메모리 위치를 확인한다.

첫 번째 데이터
pHead 포인터 변수 시작
Dummy head 관리목적 노드

연결 리스트 코딩 순서
1. 전체 리스트 출력 함수 작성
2. 새 노드 추가 함수 작성(개발에 앞서 절차를 정확히 글로 기술한다.)
3. 전체 리스트 삭제 함수 작성
4. 각 ㅎ마수를 작성할 때 마다 main()에서 테스트 코드를 실행한다
