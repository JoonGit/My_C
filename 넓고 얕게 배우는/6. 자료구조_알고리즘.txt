자료를 구조화 한다 = 자료를 적절히 정리를 한다
알고리즘 = 자료를 규칙에 따라 정리를 한다

정리를 잘해야 자료를 신속히 가져올 수 있다

정리시간이 많이 걸릴수록 쉽게 자료를 찾을수 있다

'정리'라는 행위는 상황에 맞는 적절한 규칙이 존재한다

'일정규칙으로 정리된 자료'를 자료구조 하고
구조화된 자료에서 원하는 것을 빨리 찾아내는 방법이 알고리즘 이다

스택(stack)
FILO : 가장 먼저 들어온게 가장 나중에 나온다

큐(Queue)
FIFO : 가장 먼저 들어온게 가장 먼저 나간다

선형 구조
연결리스트(Linked list)와 배열
-연결리스트 : 떨어져 있는 정보를 묶어서 정렬한다
A -> D -> C -> B 

식으로 연결되있는 구조를 찾아가다 보면 정보를 얻을수 있다

-배열 : 정보를 순서대로 나열해 있다
A -> B -> C -> D 

비선형 구조
2진 트리

비선형 구조
장점 : 효율적이고 접근성이 좋다
단점 : 비용이 많이 든다, 시간

선형 구조

단점 : 비효율 적이며, 접근성이 낮다
